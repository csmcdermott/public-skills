# Verbose-phrase substitution table (loaded on demand)

Loaded when the writer is uncertain whether a phrase is bloated. The principle: prefer the simple form when meaning is preserved.

| Verbose | Simple |
|---|---|
| are in a position to | can |
| at that point in time | then |
| at the present time | now |
| due to the fact that | because |
| in the event that | if |
| in the near future | soon |
| it is highly likely that | probably |
| it is possible that | may |
| subsequent to | after |
| the majority of | most |
| the manner in which | how |
| whether or not | whether |

The list is not exhaustive — the principle is "prefer the simple form when meaning is preserved." Verbose phrasing is almost always a tell that an AI wrote the sentence; humans contract or drop the construction.
